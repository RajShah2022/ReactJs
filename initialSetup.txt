To view the application itself:
1) Open a terminal window in the "json-server" folder and send a command of "json-server --watch db.json -d 2000 -p 3001".
2) This will start up the local json-server and ReactJS Application.
3) This application can be accessed through a browser window by entering the URL as mentioned in the terminal window under "Home".

To view source code of react components, Go to the confusion folder.
